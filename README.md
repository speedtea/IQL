🚀 IQL STORE | Premium Marketplace 2026IQL STORE adalah platform marketplace modern yang dirancang untuk memberikan pengalaman belanja yang cepat, responsif, dan interaktif. Website ini menggabungkan estetika premium dengan integrasi data real-time menggunakan teknologi web terkini.🌟 Keunggulan Utama1. Data Management Real-Time (Google Sheets CMS)Berbeda dengan web statis biasa, IQL STORE menggunakan Google Sheets sebagai Database.Update Tanpa Coding: Kamu bisa mengubah harga, nama produk, atau stok langsung dari spreadsheet tanpa menyentuh kode HTML.Skalabilitas: Memudahkan pengelolaan ratusan produk secara efisien.2. UI/UX Modern & ResponsifDibangun dengan Tailwind CSS, desain ini memberikan kesan eksklusif ala marketplace global (Amazon & Shopee style).Mobile-First Design: Dioptimalkan secara sempurna untuk pengguna smartphone.Interactive Elements: Menggunakan Swiper.js untuk banner transisi halus dan SweetAlert2 untuk modal popup yang cantik.Micro-Animations: Fitur hover effect pada kartu produk dan animasi pulse pada seksi Flash Sale meningkatkan keterlibatan pengguna.3. Sistem Keranjang Pintar (Smart Cart)Local Storage Persistence: Produk yang dimasukkan ke keranjang tidak akan hilang meskipun halaman di-refresh.Checkout WhatsApp: Integrasi langsung ke WhatsApp dengan format pesan otomatis yang rapi, memudahkan transaksi tanpa perlu sistem payment gateway yang rumit.Direct Shopee Link: Opsi bagi pengguna yang lebih memilih bertransaksi melalui platform e-commerce besar.4. Fitur Navigasi CanggihLive Search Suggestions: Pencarian produk secara instan (real-time filtering) dengan fitur auto-suggest.Category Filtering: Memungkinkan pengguna menyaring produk berdasarkan kategori (Gadget, Fashion, Elektronik, dll) secara instan.Flash Sale Countdown: Fitur urgensi belanja dengan penghitung waktu mundur otomatis hingga tengah malam.🛠️ Tech StackKomponenTeknologiFramework CSSTailwind CSS (via CDN)Library SliderSwiper.js v11Icons & PopupsFlaticon & SweetAlert2DatabaseGoogle Sheets (CSV Export)ScriptingVanilla JavaScript (ES6+)FontGoogle Fonts (Inter)📈 Fitur TambahanInstagram Feed Integration: Menampilkan produk terbaru dalam gaya grid Instagram yang modern.Floating Action Button (WA): Memudahkan pelanggan untuk bertanya kapan saja.Promo Modal: Fitur otomatis menampilkan kode promo saat pengunjung pertama kali datang (Session Storage based).Back to Top: Navigasi cepat untuk kenyamanan pengguna saat scrolling jauh.📂 Cara Penggunaan (Deployment)Upload file index.html ke GitHub Pages.Pastikan URL Google Sheets di variabel SHEET_CSV_URL sudah benar dan terpublikasi sebagai CSV.Web siap digunakan secara publik!Developed by Saka - 2026 Premium Web Project.
